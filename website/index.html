modelChecker

Reliable production ready sanity checker for Autodesk Maya

Sanity check polygon models in Autodesk Maya, and prepare your digital assets for a smooth sailing through the production pipeline.

The modelChecker is free and always will be.

Get started Download

Installation

Place the modelChecker.py file in your maya scripts directory and create a python shell button with the following code:

import modelChecker

try:
    md_win.close()
except:
     pass
md_win = modelChecker.modelChecker(parent=modelChecker.getMainWindow())
md_win.show()
md_win.raise_()
